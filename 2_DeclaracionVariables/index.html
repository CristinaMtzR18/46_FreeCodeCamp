Declaracion de variables en JavaScript

En informática, los datos es algo que significativo para el equipo. JavaScript ofrece siete tipos de datos diferentes: undefined, null, boolean, string, símbolo, número, y el object.

Por ejemplo, las computadoras distinguen entre los números, tales como el número 12, y strings, tales como "12", "perro", o "123 gatos", que son colecciones de caracteres. Las computadoras pueden realizar operaciones matemáticas con un número, pero no en una cadena.

Las variables permiten a las computadoras almacenar y manipular datos de una manera dinámica. Lo hacen mediante el uso de una "etiqueta" a punto de los datos en lugar de utilizar los datos en sí. Cualquiera de los siete tipos de datos se pueden almacenar en una variable.

Las variables son similares a las variables x e y se utiliza en matemáticas, lo que significa que es un nombre sencillo para representar los datos que queremos hacer referencia. las variables de la computadora difieren de variables matemáticas en que pueden almacenar valores diferentes en momentos diferentes.

Le decimos JavaScript para crear o declarar una variable poniendo la palabra clave var en frente de ella, así:

ourName var;

crea una variable llamada ourName. En JavaScript terminamos declaraciones con punto y coma.

Los nombres de variables pueden estar compuestos por números, letras y $ o _, pero no pueden contener espacios o comenzar con un número.

Instrucciones

Usar la palabra clave var para crear una variable llamada myName.
