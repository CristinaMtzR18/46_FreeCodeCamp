Comentarios en JavaScript

Los comentarios son líneas de código que JavaScript va a ignorar intencionadamente. Los comentarios son una gran manera de dejar notas para sí mismo y para otras personas que luego tendrán que adivinar lo que significa que el código.

Hay dos formas de escribir los comentarios en JavaScript:

Usando // dirá JavaScript para ignorar el resto del texto en la línea actual:

// Esta es una linea de código.

Usted puede hacer un comentario de varias líneas que comienzan con / * y termina con * /:

/* Esto es un
    multi-línea de comentario * /

Mejores prácticas

Como se escribe código, se debe añadir comentarios regularmente para aclarar la función de las partes de su código. Bueno comentando puede ayudar a identificar el propósito de su código, tanto para los demás y para su yo futuro.
